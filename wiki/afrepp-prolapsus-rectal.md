# AFREPP — Prolapsus Rectal

Présentation de congrès AFREPP (13 pages) couvrant la classification des prolapsus rectaux, le syndrome clinique, le diagnostic, les traitements chirurgicaux et conservateurs, et le rôle détaillé du kinésithérapeute (BDK + PEC). Inclut également les recommandations HAS sur les facteurs de risque modifiables du prolapsus génital et les données sur l'adhérence thérapeutique.

## Key Points

### Classification
- **Prolapsus rectal total** : extériorisation circulaire, 9× plus fréquent chez la femme, âge > 60 ans
- **Procidence interne bas grade** : dans le rectum, présente chez **50 % des sujets asymptomatiques**
- **Procidence interne haut grade** : dépasse la ligne pectinée
- **Prolapsus muqueux** : hémorroïdaire (étiopathogénie différente)
- **Marisques** : ne sont pas un prolapsus → ne pas réintroduire

### Syndrome clinique
Constipation (9-78 %), dyschésie (12-78 %), syndrome rectal (27-67 %), algies périnéales (22-78 %), IF selles liquides (25-78 %).

### Diagnostic clé
Examen clinique dynamique (poussée) + déféco-IRM + MAR (asynchronisme, anisme) + coloscopie.

### Traitement
- **Total** → chirurgie (ventropexie coelioscopique de référence) + rééducation si refus/récidive
- **Procidence interne** → conservateur (laxatifs + rééducation) ; chirurgie si échec

### Rôle du kiné — BDK complet
Deux profils : jeune hypertonique (↓ PIA) vs âgé hypotonique (restaurer compétence PP + abdominale). BFB ano-rectal (NICE 2021), techniques manuelles, ESF si hypotonie. Réintégration du prolapsus : manuelles/posturales/gymniques.

### Adhérence thérapeutique
> 80 % des patients ne suivent pas suffisamment leur traitement pour bénéfices optimaux (Lamouroux 2005). Facteurs : anxiété, soutien social, relation thérapeute-patient, compréhension.

### Facteurs de risque modifiables (HAS)
Poids, constipation, comportement mictionnel/défécatoire, port de charges, toux, sédentarité.
