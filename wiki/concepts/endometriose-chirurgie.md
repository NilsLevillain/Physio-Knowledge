# Endométriose — Chirurgie et Rôle du Kinésithérapeute

L'endométriose profonde peut nécessiter une chirurgie complexe selon sa localisation. Le tableau récapitulatif HAS (présenté par Agathe DEGROOTE, Masseur-Kinésithérapeute) synthétise les approches par localisation.

## Details

### Classification HAS — Recommandations par localisation

**Légende des grades :**
- NP1 Grade A (vert ✓)
- NP2 Grade B (jaune)
- NP3-4 Grade C (rouge)
- AE (accord d'experts, bleu ✓)

#### Tableau 1 : Endo pelvienne minime à légère / Vessie / Utérus / Extra-pelvienne / Nerveuse

| | Endo pelvienne minime à légère | Vessie | Utérus | Extra-pelvienne | Nerveuse |
|---|---|---|---|---|---|
| **Douleurs** | ✓ (NP1) | ✓ (AE) | | ✓ thoracique/diaphragmatique | |
| **Fertilité** | ✓ (NP1, par adhésiolyse) | | | | Incontinence/urgenturie, atonie, trouble sensibilité vésicale |
| **Complications** | Fibrose | Peu de récidives, peu de complications | Peu récidive | | |
| **Ttt complet** | ✓ (AE) | | | | |
| **Technique** | Excision / électrocoagulation | Cystectomie partielle | Hystérectomie avec ou sans annexectomie bilatérale | | Préservation nerveuse |
| **+** | | | Ttt hormonal post-chir OK | Atteinte pelvienne associée 80 % → consult gynécologue | Optimisation / neurostim per-op, neuroinstrumentation, neuroimagerie virtuelle |

#### Tableau 2 : Ovaire / Uretère / Colo-rectale endo profonde

| | Ovaire | Uretère | Colo-rectale endo profonde |
|---|---|---|---|
| **Douleurs** | | ✓ (NP2) | Oui si laparo / oui si dépasse post-op |
| **Fertilité** | | | Oui si laparo / post-op |
| **Complications** | ↑ réserve ovarienne, abandon si à risque | W pluridisciplinaire, sténose UU et aire implantation, faible | Fistules recto-vaginales, digestive, abcès pelviens, plaie vessie, plaie urétrale, faux rectum (long terme intestin) |
| **Ttt complet** | Recherche autre localisation et ttt complet | | ✓ (NP2) |
| **Technique** | Kystectomie intrapéritonéale / sclérothérapie par éthanol, électrocoagulation | Uretérolyse ou résection | Shaving, résection discoïde, résection segmentaire, pas de barrières anti-adhérentielles |
| **+** | Test de la réserve ovarienne avant / préservation fertilité | Vérification imagerie post-op | Discussion en amont sur possibles dérivation digestive temporaire has ou colostomie pour limiter risques de fistules |

### Implications pour la kinésithérapie
- L'endométriose nerveuse → atteinte du [[nerf-pudendal]] possible, techniques de neurostimulation
- L'endométriose colo-rectale profonde → risques de fistules recto-vaginales et troubles de défécation → lien avec [[rectocele]] et [[constipation-fonctionnelle]]
- Post-opératoire : [[plancher-pelvien]] souvent impliqué, douleurs pelviennes chroniques persistantes possibles

## Counter-Arguments and Gaps
- La kinésithérapie n'est pas encore intégrée dans les protocoles HAS pour l'endométriose, malgré son rôle croissant reconnu dans la gestion de la douleur et de la statique pelvienne.
- Le shaving vs la résection segmentaire reste controversé pour l'endométriose colo-rectale profonde.
